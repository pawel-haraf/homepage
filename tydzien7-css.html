Pseudoelementy & Pseudoselektory

Pseudoelement

Poprzez CSS możemy tworzyć naszą treść dzięki pseudoelementom. Jest to możliwe dzięki własności content:. Tą własność możemy używać w pseudoelementach:

  • element::before umieszczany jest przed treścią
  • element::after umieszczany jest za treścią
Pseudoklasy

Jest to modyfikator mojego elementu, którego działanie wynika z jakiegoś innego działania. Mogą to być np.:

  • hover - modyfikuje wygląd mojego elementu, w momencie najechania na ten element myszką
  • visited - linki, które już odwiedziliśmy
  • focus - definiuje oznaczenie, jakie jest wykorzystywane przy wybieraniu elementów poprzez użycie tabulatora. Funkcja przydatna również przy inputach
  • first-child - style się wykonają jeśli dany element jest pierwszym dzieckiem jakiegoś elementu
  • last-child - style się wykonają jeśli dany element jest ostatnim dzieckiem jakiegoś elementu
  • n-child - style się wykonają jeśli dany element jest n-tym dzieckiem jakiegoś elementu
  • ...
