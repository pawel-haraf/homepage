CSS: Zmienne

CSS Custom Properties

Do wykorzystania zmiennych  CSS'owych (CSS Custom Properties) wykorzystujemy pseudoselektor :root

Zmienną definiujemy poprzez dwa myślniki i nazwa zmiennej (--zmienna)

Definiujemy w :root, wartość - dowolna wartość możliwa w CSS

:root {
--small-padding: 8px;
}

Żeby pobrać wartość z tej zmiennej musimy użyć słówka var i nazwy zmiennej (var(--zmienna):

h1 {
padding: var(--small-padding);
}

Możemy zmieniać wartość zmiennych CSS'owych za pomocą JavaScriptu:

document.documentElement.style.setProperty('--zmienna', 'wartość')

Za pomocą media queries możemy nadpisywać (zmieniać) wartość naszych zmniennych

@media (min-width:756px) {
--small-padding: 16px;
}
