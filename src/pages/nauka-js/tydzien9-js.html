JavaScript: Tablice

Tablica (array) jest strukturą danych.

Jak wygląda tablica?
const hardware = ['keyboard', 'mouse', 'screen']
const news = [
{title: 'wtf',
content: 'lorem ipsum...'
},
{title: 'co ten frontend',
content: 'lorem ipsum...',
link: 'https://cotenfrontend.pl'
}
]

Każdy element w tablicy ma swój index (numer). Numerowanie elementów w tablicy zaczynamy od zera.

Jak pobrać element na danym miejscu?
tablica[index]
const myArray = [':)', ':B', ':P', ';)'];
console.log(myArray[2]); // [':P']
Ile mamy elementów w tablicy?

tablica.length - liczy od 1

console.log(myArray.length); // 4
Stringi!
const myString = 'Maciek';
console.log(myString[0]); // M
console.log(myString.length); // 6

Czyli z powyższego przykładu wynika, że stringi (tekst) jest traktowany jak tablica.

Jak dodać element (na końcu) tablicy?
tablica.push(element);
myArray.push(myString);
console.log(myArray); // [':)', ':B', ':P', ';)', 'Maciek'];
Jak dodać element (na początku)?
tablica.unshift(element)
myArray.unshift(':P'); console.log(myArray); // [':P', ':)', ':B', ':P', ';)', 'Maciek'];
Jak usunąć element z początku tablicy?
tablica.shift()
myArray.shift()
console.log(myArray); // [':)', ':B', ':P', ';)', 'Maciek'];
Jak usunąć element z końca tablicy?
tablica.pop();
myArray.pop();
console.log(myArray); // [':)', ':B', ':P', ';)'];
Jak sprawdzić czy mamy element w tablicy?
tablica.indexOf(element);
console.log(myArray.indexOf(';)')); // 3
Jak usunąć n-elementów od danej pozycji?
tablica.splice(index, liczbaElementów);
myArray.splice(1, 2); console.log(myArray); // [':)', ';)'];
Jak pobrać n-elementów od danej pozycji?
tablica.slice(index, stop);
const newArray = myArray.slice(2,4); console.log(newArray); // console.log(myArray); // [';)'];

JavaScript: Pętle

Po co nam pętle (loops)?

Pętli używamy dla rzeczy, które muszą się ileś razy wykonać, aż osiągniemy żądany efekt.

Pętla for
for (inicjalizacja; warunek stopu; wyrażenie inkrementacji) {
kod
}

i - licznik pętli

Przykład:

for (let i = 0; i < 3; i++) {
console.log(`${i} wykonanie pętli`);
}

Ważne! Jeśli zaczniemy pętle od zera, to liczba w podana w warunku stopu (i < 3) będzie określać ilość kroków.

  • i++ - postinkrementacja
  • i-- - postdekrementacja - zmniejszenie wartości
  • ++i - preinkrementacja
  • --i - predekrementacja
Pętla while
while(warunek logiczny) {
kod
}

Przykład:

let n =3;
while (n < 9) {
console.log(n);
n = n + 3;
}
console.log(n);
Pętla do while
do {
kod
} while (warunek logiczny)

Przykład:

let n = 9;
do {
console.log(n);
n = n + 3;
} while (n < 9);
console.log(n);
Pętla for in
for (własność in obiekt)

Przykład:

const person = {
name: 'Maciek',
age: 31
}
for (let propertyName in person) {
console.log(`${propertyName}: ${propertyName}: ${person[propertyName]}`);
}
Pętla for of
for (element of tablica)

Przykład:

const elements = [':B', ':P', ';)'];
for (let element of elements) {
console.log(element);
}
